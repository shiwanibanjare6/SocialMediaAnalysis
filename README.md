## Overview

This project performs **Exploratory Data Analysis (EDA)** on a social media sentiment dataset collected from multiple platforms such as **Twitter, Instagram, and Facebook**.
The goal is to analyze **user engagement, sentiment distribution, hashtags, likes, and retweets** to discover patterns in social media interactions.

This notebook focuses on **data cleaning, visualization, and insights extraction** from social media activity.

---

## Dataset

The dataset used in this project contains information such as:

* Platform name
* User posts
* Sentiment labels
* Hashtags
* Likes
* Retweets
* Timestamp (year and hour)
* Country

Example file used:

```
sentimentdataset.csv
```

## Features / Analysis Performed

### General Analysis

* Platform-wise distribution
* Overall sentiment distribution
* Hashtag frequency analysis
* Top liked posts
* Top retweeted posts

* Likes distribution by hour

### Twitter Analysis

* Sentiment distribution
* Top hashtags
* Most retweeted hashtags

### Instagram Analysis

* Sentiment distribution

### Facebook Analysis

* Sentiment distribution
* Hashtag analysis
* Most retweeted hashtags
* Most liked hashtags
